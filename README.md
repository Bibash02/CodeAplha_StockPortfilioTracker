## Stock Portfolio Tracker ##

A simple Python program that calculates the total investment value of stocks based on user input and predefined stock prices.

# Project Description #

This program allows users to:

Enter stock names

Enter quantity of shares

Calculate total investment value

Optionally save the investment summary to a .txt file

The stock prices are stored in a predefined dictionary inside the program.

# Technologies Used #

Python

Dictionary

User Input / Output

Basic Arithmetic

File Handling (Optional)

# How It Works #

The program displays available stocks.

The user enters a stock name and quantity.

The program calculates the investment for each stock.

The total investment value is displayed.

The user can choose to save the result to a file
